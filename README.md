# Techno-Sales-Data-Analysis
Interactive Excel Sales Dashboard using Power Query, Pivot Tables, Pivot Charts, KPIs, and Slicers to analyze sales, profit, orders, categories, states, and supervisor performance.

# Techno Sales Data Analysis

## Project Objective

The objective of this project is to analyze techno sales data and create an interactive dashboard in Microsoft Excel. The dashboard helps in understanding overall sales performance, profit, order status, state-wise sales, category performance, supervisor performance, and monthly sales trends.

The analysis can help the business identify high-performing states, product categories, supervisors, and months to support better business decisions.

## Dataset Used

<a href="https://github.com/Meet-Vaidya/Techno-Sales-Data-Analysis/blob/main/Techno_Sales_Data.xlsx">Techno Sales Dataset </a>

The dataset contains two sheets:

### Sales_Data

This sheet contains 5,096 sales records.

Columns used:

- Order_Number
- State_Code
- Customer_Name
- Order_Date
- Status
- Product
- Category
- Brand
- Cost
- Sales
- Quantity
- Total_Cost
- Total_Sales
- Assigned Supervisor

### State_list

This sheet contains 36 state records.

Columns used:

- State
- State_Code

## Questions / KPIs

- What is the total sales generated by the business?
- What is the total profit earned?
- What is the total number of orders?
- What is the average order value?
- Which states generate the highest sales and profit?
- What is the sales performance of each supervisor?
- Which product categories contribute the most to total sales?
- What is the monthly sales and profit trend?
- What is the distribution of orders based on order status?
- Which categories, brands, and states can be filtered for detailed analysis?

## Data Cleaning and Preparation

- Checked the dataset for missing values and incorrect data.
- Verified that the data types and formats were consistent.
- Used Power Query to merge the `Sales_Data` sheet and `State_list` sheet.
- Used `State_Code` as the common column for merging both sheets.
- Loaded the merged data back into Excel for further analysis.
- Created Pivot Tables based on the required KPIs and questions.
- Created Pivot Charts for better visualization.
- Combined all charts and KPIs into one interactive dashboard.
- Added slicers for Category, Brand, and State to make the dashboard dynamic.

## Power Query Process

The following steps were used to merge both sheets in Power Query:

1. Go to Data → Get Data → From Other Sources → Blank Query.
2. Load both `Sales_Data` and `State_list` sheets into Power Query.
3. Use the Merge Queries option.
4. Select `State_Code` as the common column in both sheets.
5. Choose the required join type.
6. Load the merged result back into Excel.

## Dashboard Features

The dashboard includes the following KPIs:

- Total Sales
- Total Profit
- Total Orders
- Average Order Value

The dashboard also includes the following analysis:

- Order Status Analysis
- State-wise Sales and Profit Analysis
- Supervisor Performance Analysis
- Sales by Category
- Monthly Sales and Profit Trend
- Category Filter
- Brand Filter
- State Filter

## Project Insights

- The dashboard provides an overview of total sales, total profit, total orders, and average order value.
- State-wise analysis helps identify states that contribute the highest sales and profit.
- Supervisor performance analysis helps compare supervisors based on total sales and total orders.
- Category-wise analysis helps identify product categories that generate higher sales.
- Monthly trend analysis helps understand changes in sales and profit throughout the year.
- Order status analysis helps track orders that are delivered, shipped, processing, or placed.
- Slicers allow users to filter the dashboard by category, brand, and state for detailed analysis.

## Final Conclusion

This project shows how Microsoft Excel can be used to clean, merge, analyze, and visualize sales data.

Power Query was used to merge the sales data with state information using the `State_Code` column. Pivot Tables, Pivot Charts, KPIs, and slicers were used to create an interactive dashboard.

The dashboard helps the business understand sales performance across different states, categories, brands, supervisors, and months. It can support better decision-making by identifying high-performing areas and improving sales strategies.
